# BiblioSystem
Sistema de gestión para biblioteca universitaria. Funciona 100% en el navegador usando `localStorage`, sin backend ni instalación.

## Archivos
index.html   — estructura de la app
styles.css   — estilos y paleta de colores
app.js       — toda la lógica del sistema


## Funcionalidades
- Registro de libros (código, título, autor, categoría, ejemplares)
- Registro de usuarios (estudiantes, docentes, administrativos)
- Control de préstamos y devoluciones
- Alertas automáticas de vencimiento
- Reportes y estadísticas
- Exportación a CSV

## Uso
Abre https://axelmarroquinn.github.io/sistema-biblioteca/
- Los datos se guardan en `localStorage` del navegador. 
- Para limpiar todo: Reportes → *Limpiar todos los datos*.
